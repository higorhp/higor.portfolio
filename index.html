Projetos

Projeto Jarvis

Início:Em Desenvolvimento

Estou desenvolvendo um assistente virtual inspirado no Jarvis, focado em automação e comandos de voz. O projeto inclui funcionalidades como ativação por comando, controle de dispositivos como lâmpadas inteligentes, ajuste de volume do Windows e execução de tarefas rotineiras. O código foi modularizado para facilitar a manutenção, e estou integrando novos recursos como verificação de temperatura atual e respostas inteligentes.

Projeto Portfólio

Projeto Consolidador de Planilhas Automático

Início:Concluído

Este projeto visa automatizar o processo de consolidação de planilhas em um único arquivo. Ele utiliza Python para integrar diferentes fontes de dados, realizar cálculos e gerar relatórios automaticamente, economizando tempo e reduzindo erros humanos no processo.
